# セットアップガイド

このガイドでは、Fudebakoを最初から使えるようになるまでの手順を説明します。

---

## 前提条件

- **Claude Proアカウント**（月額$20）以上 — [claude.ai](https://claude.ai) で登録
- **Claude Desktop（Cowork対応版）**がインストール済み — [claude.ai/download](https://claude.ai/download)
- **Windows 10以降** または **macOS**（Cowork対応版）

まだClaude Desktopをインストールしていない場合は、上記リンクからダウンロードしてください。

---

## 全体の流れ

```
Step 1: ダウンロード          ← このリポジトリを手元に
Step 2: セキュリティ設定      ← AI学習OFF（最重要）
Step 3: フォルダ作成          ← 作業環境を作る
Step 4: グローバル設定        ← AIのルールを設定
Step 5: キャラクターシート    ← クリエイターが書く（任意）
Step 6: 記憶の引き継ぎ       ← ChatGPT/Geminiから移行（任意）
Step 7: 初回起動              ← Coworkを開いてプロンプトを貼る
```

所要時間の目安：
- Step 1〜4（設定する人）: **10分**
- Step 5〜6（クリエイター）: **20〜30分**（任意）
- Step 7（クリエイター）: **10分**

---

## Step 1: ダウンロード

### GitHubから

```bash
git clone https://github.com/fbkatz/fudebako.git
```

### ZIPダウンロード

GitHubページ右上の「Code」→「Download ZIP」でもOKです。

---

## Step 2: セキュリティ設定（最重要）

**⚠️ これが終わるまで他のステップに進まないでください。**

1. Claude Desktop を開く
2. 左下のアカウントアイコンをクリック
3. 「設定」を開く
4. 「プライバシー」→「プライバシー設定」を選ぶ
5. 「Claudeの改善にチャットやコーディングセッションの使用を許可する」を探す
6. **OFF（グレー）にする**
7. 設定を保存する

**確認**: トグルがOFF（グレー）になっていることを目視で確認してください。

この設定により：
- 会話の内容がAIの学習に使われなくなります
- ファイルの内容も学習対象外になります
- データ保持期間が30日になります（30日後に自動削除）

詳細 → `guide/security.md`

---

## Step 3: フォルダ作成

### Windows の場合

`setup/create-folders.bat` をダブルクリック。

デフォルトでは `C:\Users\（ユーザー名）\Fudebako` にフォルダが作られます。
別の場所に作りたい場合は、batファイルをテキストエディタで開いて
`BASE_DIR` の行を書き換えてください。

### Mac の場合

ターミナルを開いて：

```bash
chmod +x setup/create-folders.sh
./setup/create-folders.sh
```

デフォルトでは `~/Fudebako` にフォルダが作られます。

### 作成されるフォルダ

```
Fudebako/
├── Claude Context/   ← AI設定ファイル置き場（自動読み込み）
├── 素材/             ← 原稿・参考資料を入れる場所
├── 成果物/           ← AIが作ったファイルの出力先
└── 作業中/           ← 一時ファイル用
```

---

## Step 4: グローバルインストラクション設定

1. Claude Desktop を開く
2. 左メニューから「Cowork」モードにする
3. 設定 → Cowork → グローバルインストラクション を開く
4. `setup/global-instructions.md` を開く
5. 「---」の行 **より下の部分だけ** をすべてコピー
6. グローバルインストラクションの入力欄に貼り付け
7. 保存する

---

## Step 5: 相棒のキャラクターシート（任意）

**ここからはクリエイター本人の作業です。**

`guide/character-sheet.md` を開いて、自分の手で書いてください。
AIに書かせるものではありません。

書く内容：
- AIにどんな性格でいてほしいか
- 口調、褒め方、ダメ出しの仕方
- やってほしいこと／やってほしくないこと
- 自分の癖（機嫌の良し悪しのサイン等）

空欄があっても構いません。後から書き足せます。

書いたら `Fudebako/Claude Context/` フォルダに入れてください。
ファイル名は `character-sheet.md` のままでOKです。

---

## Step 6: 既存AIからの記憶引き継ぎ（任意）

ChatGPTやGeminiを使っていた場合、育ててきた「あの子」の記憶を引き継げます。

1. `guide/migrate-from-other-ai.md` を開く
2. 中に書かれたプロンプトをコピー
3. ChatGPTまたはGeminiの、一番長く使っている会話に貼る
4. 出力されたテキストを保存
5. `Fudebako/Claude Context/` フォルダに入れる

複数のAIを使っていた場合は、それぞれで実行してOKです。

**注意**: キャラクターシートと抽出結果が矛盾する場合、キャラクターシートが優先されます。
あなたが自分の手で書いたものが正です。

---

## Step 7: 初回起動

1. Claude Desktop を開く
2. 左メニューから「Cowork」を選ぶ
3. 「フォルダを選択」で `Fudebako` フォルダを選ぶ
4. `prompts/first-session.txt` の中身を **すべてコピー**
5. Coworkの入力欄に貼り付けてEnterを押す
6. AIの質問に1つずつ答えていく

AIが以下を順番にやってくれます：
- あなたへのヒアリング（名前、仕事、好み）
- 設定ファイルの自動生成（about-me.md等）
- キャラクターシートの読み込み（あれば）
- フォルダ構成の相談
- 使い方の案内

---

## 動作確認

初回起動後、以下を試して正しく動くか確認してください：

```
✅ 「私の名前は？」 → あなたの名前を答える
✅ 「素材フォルダの中身を見せて」 → フォルダ内を一覧表示する
✅ 「Claude Contextの中身を見せて」 → 設定ファイルの一覧が出る
```

## トラブルシューティング

### Coworkが起動しない
- Claude Desktopが最新版か確認してください
- 再起動してみてください

### フォルダにアクセスできない
- Coworkで正しいフォルダを選択しているか確認
- フォルダのパスにスペースや特殊文字が含まれていないか確認

### AIがキャラクターシートを読んでくれない
- ファイルが `Claude Context` フォルダの直下にあるか確認
- ファイルの拡張子が `.md` か `.txt` か確認
- Coworkを再起動して、フォルダを再選択してみてください

### AIが日本語で返事をしない
- グローバルインストラクションが正しく設定されているか確認
- 「日本語で答えてください」と伝えてみてください

### 既存AIの抽出プロンプトがうまく動かない
- 会話が短い場合、十分な情報が出ないことがあります
- 一番長く使っている会話で試してください
- ChatGPTの場合、メモリ機能がONだとより良い結果が出ます

---

## 次のステップ

セットアップが終わったら：

1. `guide/security.md` を読む — 何が守られているかを理解する
2. 素材フォルダにファイルを入れてみる — 原稿、資料、なんでもOK
3. 普通の日本語で話しかける — 「この資料を読んで〇〇して」
4. キャラクターシートを育てる — 「やっぱりこうがいい」と思ったら書き直す

**あなたの仕事は書くことです。環境のことは忘れてください。**
