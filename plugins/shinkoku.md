# 確定申告自動化プラグイン（フリーランス向けオプション）

> ⚠️ **この設定はMCPサーバーの技術的知識が必要です。**
> エンジニアや技術担当者向けです。一般のクリエイターの方は、担当のIT管理者に相談してください。

## 概要

フリーランスのクリエイター向けの確定申告自動化プラグインです。
レシート撮影 → OCR → 仕訳 → 決算書 → e-Tax入力 まで自動化できます。

**対象**: 個人事業主として確定申告をしているクリエイター

## 対応する確定申告

| 項目 | 対応 |
|------|:---:|
| 個人事業主（青色申告） | ✅ |
| 源泉徴収票の処理 | ✅ |
| ふるさと納税 | ✅ |
| 医療費控除 | ✅ |
| 消費税申告 | ✅ |
| e-Tax連携 | ✅ |

| 項目 | 対応 |
|------|:---:|
| 株式・投資関連 | ❌ |
| 不動産所得 | ❌ |
| 白色申告 | ❌ |

## 設定する人がやること

### 1. プラグインのインストール
1. Claude Desktop → Coworkモード
2. プラグイン > 個人用 > GitHub からマーケットプレイスを追加
3. `kazukinagata/shinkoku` を入力して追加
4. 表示される shinkoku プラグインをインストール

### 2. フォルダの追加
`create-folders` スクリプトに以下を追加するか、手動で作成：

```
作業フォルダ/
└── 確定申告/
    └── 2025年分/
        ├── レシート/
        ├── 源泉徴収票/
        ├── 請求書/
        ├── 控除証明書/
        ├── ふるさと納税/
        └── 医療費/
```

### 3. グローバルインストラクションに追加
以下を既存のグローバルインストラクションに追記：

```
## 確定申告の安全ルール
- e-Taxの最終送信ボタンは絶対にAIが押さない（人間のみ）
- 税額の計算結果は必ず人間に確認を求める
- マイナンバー等の個人情報は最大限の注意を払う
- 判断に迷う場合は税理士への相談を推奨する
```

## クリエイターの使い方

### 年間を通してやること
- レシート・領収書をスマホで撮影
- 該当フォルダに入れる

### 申告時（2〜3月）
1. Coworkで「確定申告やりたい」と入力
2. AIの質問に答える
3. レシート読み取り → 仕訳 → 決算書 → e-Tax入力まで自動
4. **最終送信ボタンだけ自分で押す**

### 注意
- AIの計算結果は提出前に必ず自分で確認
- 判断に迷ったら税理士に相談
- 送信ボタンはAIに押させない（安全ルール）
