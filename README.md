# Fudebako 筆箱

**文章を書く人が、安心してAIと仕事をするためのスターターキット**

[![License: CC0-1.0](https://img.shields.io/badge/License-CC0_1.0-lightgrey.svg)](http://creativecommons.org/publicdomain/zero/1.0/)

---

## これは何ではないか

- AIに文章を書かせるツールではありません
- 「AI活用で生産性10倍！」を謳うものではありません
- プロンプトエンジニアリングのテクニック集ではありません

## これは何か

**文章を書くクリエイターが、AIを「雑務担当の相棒」として安全に使うための環境キット**です。

作家、脚本家、ライター、編集者、コピーライター、ゲームシナリオライター。
文章で食べている人たちの多くは、AIに対してこう思っています：

- 「自分の作品が学習素材にされるのが怖い」
- 「使ってみたいけど、何をどう設定すればいいかわからない」
- 「書くのは自分の仕事。でも、調べ物や整理は誰かにやってほしい」
- 「ChatGPTで育てた『あの子』の記憶を引き継ぎたい」
- 「クライムサスペンスやベッドシーンを書いてたらBANされない？」

このキットは、これらの不安に一つずつ答えます。

## 思想

```
創作は人間がやる。
雑務はAIがやる。
創作物は外に出さない。
判断はクリエイターがする。
```

Fudebako（筆箱）は、筆を入れる箱です。
筆で何を書くかは、あなたが決めます。

## 誰が使うか

このキットは **2人1組** で使います。

| 役割 | 誰 | やること |
|-----|-----|----------|
| **設定する人** | ITに詳しい家族、友人、同僚、マネージャー | セキュリティ設定、フォルダ作成、プラグイン導入 |
| **使う人** | 文章を書くクリエイター | キャラクターシートを書く、素材を入れる、AIと対話する |

「設定する人」がいない場合でも、ガイドに沿って自分でセットアップできます。

## セットアップ（5分）

**👉 [詳細なセットアップガイドはこちら → SETUP.md](SETUP.md)**

以下は概要です。手順の詳細はSETUP.mdを参照してください。

### 設定する人がやること

```
1. このリポジトリをダウンロードする
2. setup/security-checklist.md を上から順に全部やる ← 最重要
3. setup/create-folders.bat（またはsh）を実行する
4. setup/global-instructions.md の内容をCoworkの設定に貼る
5. クリエイターにガイドを渡す
```

### クリエイターがやること

```
1. guide/security.md を読む（不安を解消する）
2. guide/character-sheet.md を自分の手で書く
3. guide/getting-started.md に沿ってCoworkを起動する
4. prompts/first-session.txt をCoworkに貼る
5. AIの質問に答えていく
```

## よくある質問

### 「AIに書かせないのに、AIを使う意味あるの？」

あります。クリエイターの仕事の半分以上は「書く」以外です。

- 原作100ページの登場人物一覧を作る
- 第3話と第7話でキャラの性格がブレてないか確認する
- 20話分の伏線回収状況を表にまとめる
- 時代考証の事実確認をする
- 前の稿と今の稿の差分を見せる
- レシートを仕訳して確定申告の準備をする

これらは「書く才能」とは関係のない作業です。
ここをAIに任せて、書く時間を増やすのがFudebakoの目的です。

### 「本当に学習に使われないの？」

`guide/security.md` に、Anthropicの公式ポリシーに基づいた詳細な説明があります。
結論だけ言えば：設定をOFFにすれば使われません。

### 「クライムサスペンスやベッドシーンでBANされない？」

`guide/security.md` の「脚本の内容でBANされない？」セクションを読んでください。
結論：フィクションの創作でBANされることはありません。
「拒否」（このシーンは書けません）はありますが、それはアカウント停止とは別物です。

### 「ChatGPTで育てたAIの記憶を引き継げる？」

`guide/migrate-from-other-ai.md` にプロンプトがあります。
既存AIに貼ると、性格・記憶・暗黙のルールを定型フォームで出力します。
