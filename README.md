# Fudebako 筆箱

**文章を書く人が、安心してAIと仕事をするためのスターターキット**

[![License: CC0-1.0](https://img.shields.io/badge/License-CC0_1.0-lightgrey.svg)](http://creativecommons.org/publicdomain/zero/1.0/)

---

## これは何ではないか

- AIに文章を書かせるツールではありません
- 「AI活用で生産性10倍！」を謳うものではありません
- プロンプトエンジニアリングのテクニック集ではありません

## これは何か

**文章を書くクリエイターが、AIを「雑務担当の相棒」として安全に使うための環境キット**です。

作家、脚本家、ライター、編集者、コピーライター、ゲームシナリオライター。
文章で食べている人たちの多くは、AIに対してこう思っています：

- 「自分の作品が学習素材にされるのが怖い」
- 「使ってみたいけど、何をどう設定すればいいかわからない」
- 「書くのは自分の仕事。でも、調べ物や整理は誰かにやってほしい」
- 「ChatGPTで育てた『あの子』の記憶を引き継ぎたい」
- 「クライムサスペンスやベッドシーンを書いてたらBANされない？」

このキットは、これらの不安に一つずつ答えます。

## 思想

```
創作は人間がやる。
雑務はAIがやる。
創作物は外に出さない。
判断はクリエイターがする。
```

Fudebako（筆箱）は、筆を入れる箱です。
筆で何を書くかは、あなたが決めます。
