# セキュリティ設定チェックリスト（設定する人用）

クリエイターにCoworkを渡す **前に** 必ず完了させること。
これがキットの中で **最優先** の手順。

---

## チェックリスト

### □ 1. AI学習をOFFにする（最重要）
1. Claude Desktop を開く
2. 左下のアカウントアイコン → 設定
3. 「プライバシー」→「プライバシー設定」
4. 「Claudeの改善にチャットやコーディングセッションの使用を許可する」→ **OFF**
5. 設定を保存

**確認方法**: 設定画面でトグルがOFF（グレー）になっていること

### □ 2. フォルダアクセスを限定する
- Coworkで選択するフォルダは作業フォルダのみ
- ホームディレクトリ全体やドライブ全体を選択しない
- 他のフォルダを追加する場合はクリエイターに確認してから

### □ 3. グローバルインストラクションを設定する
- `setup/global-instructions.md` の内容を設定に貼る

### □ 4. フォルダを作成する
- `setup/create-folders.bat`（Windows）または
  `setup/create-folders.sh`（Mac/Linux）を実行

### □ 5. 動作確認する
- Coworkを起動して作業フォルダを選択
- 「素材フォルダの中身を見せて」と打って、フォルダアクセスが正しく動くか確認
- 他のフォルダ（例: ホームディレクトリ）にアクセスできないことを確認

### □ 6. セキュリティ資料をフォルダに入れる
- `guide/security.md` を作業フォルダの直下にコピー
- クリエイターがいつでも読み返せるようにしておく

---

## クリエイターに渡すとき

### 渡す順番（これ重要）

1. **最初に**「創作物の保護について」（security.md）を見せる
   - 「学習に使われない」「フォルダ外にはアクセスできない」を説明
   - 不安や質問があれば全部ここで解消する

2. **次に**「相棒のキャラクターシート」（character-sheet.md）を渡す
   - 「これを自分で書いて」と伝える。AIに書かせないこと
   - 空欄があってもOK、あとから足せばいいと伝える

3. **その次に**「はじめかた」（getting-started.md）を渡す
   - セキュリティの不安がない状態で始められるようにする

4. **最後に**「初回プロンプト」（first-session.txt）を案内する

### 伝えるべきこと
- 「学習OFFは設定済み。触らなくていい」
- 「原稿も資料も、この設定なら外に出ない」
- 「不安なときはシークレットモードを使えばさらに安全」
- 「設定がおかしくなったら自分に言って」

### 伝えなくていいこと
- 技術的な詳細（サンドボックス、仮想マシン等）
- 30日保持期間の仕組み
- Anthropicのプライバシーポリシーの細かい条文
- 他社AIとの詳細な比較
